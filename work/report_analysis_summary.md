# 季报和年报统计报表补全说明

## 参考文件

- 年报：`01_物探检测项目部年报（2025年）（NB-WTJC-2025）.docx`
- 季报：`物探检测项目部2026年第三季度试验检测报告.docx`

## 参考表格口径

- 锚杆无损检测：按单位工程、分部工程或工程部位统计施工数量、检测数量、抽检比例、I-IV 类成果和合格率。
- 锚杆拉拔试验：按工程部位统计施工数量、检测数量、检测比例、锚杆直径、设计拉拔力、实测最小值、实测最大值和合格率。
- 回填灌浆质量检测：按工程部位统计检查孔编号、检查桩号、孔口高程、设计压力、实测压力、设计要求、初始 10min 注浆量和检测结果。
- 钻孔摄像、钻孔成像：按工程单位、分部工程、钻孔编号、检测里程和检测情况统计。
- 锚索多循环张拉、预应力锚杆张拉：按单元工程、分部工程、施工数量、检测数量、编号、长度和检测结果统计。
- 桩身完整性检测：按单元工程、分部工程、工程部位、检测根数、综合评判类别和检测结果统计。
- 洞室松弛圈检测、弹性波检测：按工程部位汇总检测孔数或组数、检测类型、检测结果和报告编号。
- 已开展工作完成合同量情况：按标段、委托类别和检测项目统计工程总量、累计完成量、本季度或年完成量、累计完成比例。

## 覆盖与缺口

现有“检测数据统计”已经能生成月报第五章口径的各检测项目统计表，并支持 Word、Excel 导出；“单位工程统计”已经能在页面按单位工程查看覆盖情况。

本次补齐的缺口是：季报、年报选择后自动追加综合统计表，包括合同量完成情况、单位工程检测覆盖情况和各检测项目正文统计表，并让这些表同时进入页面展示、Word 导出和 Excel 导出。

## 数据来源

所有统计均来自当前台账数据库 `outputs/ledger_system/ledger_system.db` 中已导入的台账工作表，不从参考年报或季报复制数据。参考文件仅用于确定表格字段和统计口径。
